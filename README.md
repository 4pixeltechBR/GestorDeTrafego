# GestorDeTrafego
Projeto Copiloto de Tráfego Pago com IA — Open Source para pequenos empresários brasileiros

Criado com Google Antigravity (Gemini 3.1 e Claude)

A ideia do projeto é desenvolver um agente de gestão de tráfego pago para auxiliar pequenos empresários a alavancar seus negócios usando inteligência artificial como copiloto. Visando a redução de custos, pensei em imprensar uma cascata com os melhores modelos de LLMs de APIs com free tiers disponíveis, como Gemini, Groq, Nvidia, Mistral e OpenRouter.
